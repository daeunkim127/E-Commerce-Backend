# E-Commerce-Backend


## Description
A back end of E-commerce site using express.js, sequelize, and mysql database.

## Installation
`Sequelize` and `mysql` were installed.

## Usage
When `node server.js` command is running, the Sequelize models are synced to the My SQL.
